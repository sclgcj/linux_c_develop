1、	线性表是n个数据元素的有限序列
	
2、	线性表顺序表示
	线性表的顺序表示指的是用一组地址连续的存储大暖依次存储线性表的数据元素。

3、	线性表的链式表示
	线性表链式存储结构的特点是：用一组任意的存储单元存储线性表的数据元素

4、	文件说明
	linear_table.h			定义了线性表的一组操作
	sqlist.c sqlist.h		定义并实现了线性表的顺序表示
	linklist.c linklist.h	定义并实现了线性表的链式表示
